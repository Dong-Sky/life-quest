# Life Quest

一个通用的轻度游戏化个人经营系统。

用户定义自己的人生领域、主线、副本、任务与现实奖励；系统负责一致的 XP、金币、等级、结算与周期反馈。

## 当前阶段

Sprint 0：建立可运行的 Web 项目骨架与浅色工作台体验。

## 开发原则

- 先完成一个可验证的用户闭环，再扩展功能。
- 用户定义内容；系统计算奖励。
- 私密优先；AI 建议必须由用户确认。
- 每项业务规则都应可测试、可解释、可追溯。
