# 奖励规则 v1

## 原则

用户只描述任务属性；系统负责计算奖励。结算时必须保存 XP、金币、计算明细和规则版本，历史记录不得因未来规则调整而重算。

## XP

`XP = round(基础 XP × 难度系数 × 重要度系数 × 心理阻力系数)`

| 任务类型 | 基础 XP |
|---|---:|
| 微型 | 5 |
| 普通 | 10 |
| 重点 | 20 |
| 里程碑 | 50 |
| Boss | 100 |

| 难度 | 系数 |
|---|---:|
| 简单 | 0.8 |
| 普通 | 1.0 |
| 困难 | 1.3 |
| 极难 | 1.6 |

| 重要度 | 系数 |
|---|---:|
| 日常维护 | 0.8 |
| 有帮助 | 1.0 |
| 推动目标 | 1.3 |
| 关键结果 | 1.6 |

| 心理阻力 | 系数 |
|---|---:|
| 无明显阻力 | 1.0 |
| 有些不想做 | 1.1 |
| 明显拖延 | 1.3 |
| 长期回避 | 1.5 |

## 金币

- XP 少于 10：0 金币。
- 其他任务：`floor(XP / 10)` 金币。

## 等级

角色在 `100 × (等级 - 1)²` 累计 XP 时进入该等级。

- Lv.1：0 XP
- Lv.2：100 XP
- Lv.3：400 XP
- Lv.4：900 XP

当前规则版本：`v1`。
