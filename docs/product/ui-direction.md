# UI 方向：浅色工作台 v1

## 目标

界面应像一套安静、可靠的个人工作台：清晰、轻量、可长时间使用。参考现代 AI 产品的层级与信息密度，但不复制任何品牌标识、图标或专有素材。

## 视觉语言

- 浅暖灰页面背景，白色内容面板，细灰色分隔线。
- 高可读性的近黑正文与中性灰辅助信息。
- 单一克制的强调色，用于主操作、进度与成功状态。
- 圆角适中、阴影极轻，避免仪表盘式“卡片海”。
- XP、金币与等级作为辅助反馈，不能盖过任务本身。

## 布局

- 左侧可折叠导航：今日、任务、副本、主线、周结算。
- 顶部保持轻量，只展示当前上下文和关键状态。
- 主内容区使用单列优先的阅读与操作流；窄屏时导航收起。
- 首页优先呈现“今日三件事”、角色状态和正在推进的副本。

## 首版组件

- 侧栏导航
- 页面标题与上下文操作区
- 任务列表和任务创建表单
- 角色状态卡（等级、XP、金币、距下一等级）
- 完成任务后的简短结算反馈
- 空状态、加载状态、错误状态

## 可访问性

- 正文与背景保持足够对比度。
- 所有核心操作可键盘访问。
- 状态不只依赖颜色表达。
- 移动端优先保证创建、查看和完成任务三个操作。
