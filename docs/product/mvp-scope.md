# MVP 范围

## 首个可体验闭环

用户创建任务 → 选择任务属性 → 完成任务 → 系统计算 XP 与金币 → 角色状态即时更新。

## 本阶段包含

- 单用户本地演示数据
- 任务创建、列表与完成
- 统一的 XP、金币和等级计算
- 完成记录与奖励流水模型
- 角色状态卡
- 浅色工作台基础界面
- 自动化测试与开发文档

## 暂不包含

- 登录、多人协作与共享副本
- 真实云端数据库与生产级权限
- 重复任务、商城、成就、日历同步
- AI 自动拆解或主动提醒
- 公开排行榜、支付、原生移动应用

## 后续顺序

1. 奖励引擎与任务结算
2. 人生领域与主线
3. 副本与里程碑
4. 重复任务与柔性连胜
5. 奖励商城与周复盘
6. 账户、多用户隔离与共同副本
7. AI 建议与自动化
