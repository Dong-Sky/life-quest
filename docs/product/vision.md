# Life Quest 产品愿景

## 一句话定义

Life Quest 是一套通用的、轻度游戏化的个人经营系统：用户定义自己的人生内容，系统提供一致的任务反馈、经验、金币、等级和周期复盘。

## 核心闭环

目标与项目 → 可执行任务 → 完成记录 → XP 与金币结算 → 今日反馈 → 周度复盘 → 调整下一步。

## 产品原则

1. **人生个性化，规则标准化。** 用户决定目标、主线、项目、任务和现实奖励；系统计算 XP、金币、等级与成就。
2. **同一事实只记录一次。** 页面只从底层记录派生，不要求用户重复维护。
3. **先帮助行动，再提供游戏感。** 游戏化是反馈层，不增加维护负担。
4. **私密优先。** 个人数据默认不可见；未来的分享与共同副本必须显式开启。
5. **AI 提供建议，不替用户决定人生。** AI 生成的任务、复盘或建议必须由用户确认后写入。

## 术语

- **领域**：人生中的一个关注面，例如健康或学习。
- **主线**：持续数月的长期方向。
- **副本**：有明确胜利条件和终点的项目。
- **任务**：当前可完成的行动。
- **Boss 战**：高价值、高压力或关键节点任务。
- **结算**：完成任务后写入奖励流水并更新角色状态。

## V1 验证目标

用户可以在三分钟内创建并结算一项任务，理解自己为什么获得这些奖励，并愿意在一周后继续使用。
