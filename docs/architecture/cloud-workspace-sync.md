# 云端工作台同步

本功能把登录用户的 Questline 工作台状态保存到 Supabase。

## 行为

- 第一次登录：将浏览器中已有的工作台内容迁入云端。
- 之后登录同一账号：优先读取云端内容。
- 每次创建、编辑或结算后：自动保存到云端。
- 新设备只需登录同一用户名和密码，刷新后即可获得同一份数据。

## 部署后设置

在 Supabase SQL Editor 运行：

`supabase/migrations/202607150002_workspace_states.sql`

运行成功后刷新 Questline。若页面提示“云端工作台尚未准备好”，通常表示这份迁移尚未执行。

## 当前边界

本轮同步的是个人工作台。共同副本、好友和成员权限将在下一阶段使用独立的数据结构实现。
