# Sprint 3B — 副本里程碑

## 目标

让用户把一个副本拆成若干可见的阶段，并在推进过程中记录阶段完成情况。

## 已包含

- 在任意副本内新增文字里程碑
- 标记一个里程碑为已完成
- 显示该副本的里程碑完成数量
- 里程碑与任务奖励系统分离：完成里程碑不发 XP 或金币
- 浏览器刷新后保留里程碑记录

## 本轮不做

- 编辑、删除或重新打开里程碑
- 里程碑日期、负责人、依赖或提醒
- 将里程碑完成度混入任务完成度
- 副本自动关闭、结算或发奖

## 验收路径

1. 创建或打开一个副本。
2. 点击“新增里程碑”，输入一个阶段目标并添加。
3. 添加 2–3 个里程碑，确认显示完成数量。
4. 点击“完成阶段”，确认该里程碑划线并显示“已完成”。
5. 确认 XP、金币和最近结算没有变化。
6. 刷新页面，确认里程碑仍存在。
