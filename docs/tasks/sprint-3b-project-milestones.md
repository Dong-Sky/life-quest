# Sprint 3B — 任务驱动的副本里程碑

## 目标

让里程碑成为一组任务的阶段性结果：所有关联任务完成结算后，里程碑自动完成。

## 已包含

- 在任意副本内新增里程碑
- 创建或编辑里程碑时选择该副本中的任务
- 里程碑实时显示关联任务的结算进度
- 所有关联任务完成后，里程碑自动显示“自动完成”
- 新任务可在之后通过“编辑里程碑”加入阶段
- 里程碑不单独发 XP 或金币，奖励只来自任务结算
- 浏览器刷新后保留任务关联和里程碑记录

## 本轮不做

- 编辑、删除或重新打开任务本身
- 里程碑日期、负责人、依赖或提醒
- 自动关闭副本、结算副本或发放副本奖励
- 周期任务关联副本

## 验收路径

1. 创建一个副本，并在任务页创建两项普通任务关联到它。
2. 回到副本页新增里程碑，勾选这两项任务。
3. 结算其中一项任务，确认里程碑显示“已结算 1 / 2 项关联任务”，仍是推进中。
4. 结算第二项任务，确认里程碑自动显示“自动完成”。
5. 确认 XP 与金币只因任务结算发生变化。
6. 新增或调整任务后，点击“编辑”里程碑更新关联任务。
