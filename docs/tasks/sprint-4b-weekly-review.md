# Sprint 4B — 本周结算

## 目标

把本周真实执行结果呈现为清晰的副本结算，强化奖励目标，并将下周最重要的行动直接创建成完整任务。

## 已包含

- 自动统计当周已结算任务、XP、金币与待推进任务
- 展示当周每项已结算任务及其奖励
- 展示本周已兑换的真实奖励
- 奖励可加入愿望单并置顶；最新置顶项为“当前最想换”
- 周结算显示当前首愿望与距兑换所差金币
- 输入最多三项下周行动，并分别设定任务类型、难度、重要度、心理阻力和可选主线
- 一次点击将非空行动创建为真实任务
- 同一周只能创建一次下周任务，避免重复生成
- 周结算本身不额外发放金币或经验

## 本轮不做

- 自由文本复盘、胜利/阻碍保存或复盘历史
- 下周任务的副本关联与周期任务设定
- AI 分析、主动建议或日历同步
- 时间投入统计、延期判定和目标权重分析
- 团队或伴侣共享结算

## 验收路径

1. 完成一项任务后打开“周结算”，确认任务数、XP、金币和战报条目一致。
2. 在“奖励商城”创建奖励并加入愿望单，确认它显示为首愿望和剩余金币。
3. 兑换一项奖励，回到周结算，确认它出现在“本周已兑换奖励”。
4. 为一至三项下周行动选择任务属性与可选主线，点击“创建下周任务”。
5. 前往任务清单，确认这些行动保留了所选属性；回到周结算确认本周不会重复创建。
