# Sprint 4B — 周结算

## 目标

每周将真实执行数据转化为简洁的复盘，并留下下周最重要的三件事。

## 已包含

- 自动统计当周已结算任务、XP、金币与待推进任务
- 记录本周最重要的胜利与最大阻碍
- 记录下周最多三项重点
- 同一周重复保存时更新同一份周结算
- 查看最近周结算摘要
- 周结算本身不额外发放金币或经验

## 本轮不做

- AI 分析、主动建议或日历同步
- 时间投入统计、延期判定和目标权重分析
- 团队或伴侣共享复盘
- 周结算奖励与连续周复盘成就

## 验收路径

1. 完成一项任务后打开“周结算”。
2. 确认任务数、XP 与金币与当周结算一致。
3. 写下胜利、阻碍和下周三件事并保存。
4. 刷新后确认内容仍然存在。
5. 再次保存修改，确认不会生成同一周的重复记录。
