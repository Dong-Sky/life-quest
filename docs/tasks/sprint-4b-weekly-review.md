# Sprint 4B — 本周结算

## 目标

把本周真实执行结果呈现为清晰的副本结算，并将下周最重要的行动直接创建成任务。

## 已包含

- 自动统计当周已结算任务、XP、金币与待推进任务
- 展示当周每项已结算任务及其奖励
- 输入最多三项下周行动
- 一次点击将非空行动创建为真实任务
- 同一周只能创建一次下周任务，避免重复生成
- 周结算本身不额外发放金币或经验

## 本轮不做

- 自由文本复盘、胜利/阻碍保存或复盘历史
- AI 分析、主动建议或日历同步
- 时间投入统计、延期判定和目标权重分析
- 团队或伴侣共享结算

## 验收路径

1. 完成一项任务后打开“周结算”。
2. 确认任务数、XP、金币和战报条目与当周结算一致。
3. 输入一至三项下周具体行动，点击“创建下周任务”。
4. 前往任务清单，确认这些行动已经成为待完成任务。
5. 回到周结算，确认本周不会再次创建重复任务。
