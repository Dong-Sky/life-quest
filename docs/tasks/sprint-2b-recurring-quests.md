# Sprint 2B — 周期任务

## 目标

支持用户把任务设为每天或每周重复，适用于运动、阅读、复盘等长期习惯。

## 已包含

- 创建任务时可开启“设为周期任务”
- 每天：当前日期内最多完成一次，次日自动重置
- 每周：设置每周 1–7 次目标，周一开始新的周期
- 任务页与今日页显示当前周期进度
- 每一次记录都会通过既有奖励引擎结算，并写入奖励流水
- 当前周期完成后不能继续领取奖励
- 使用浏览器日期自动重置，不需要后台定时服务

## 本轮不做

- 每月、自定义间隔或指定星期几
- 假期、暂停、补签与提醒
- 周期任务的经验递减或每日分类上限
- 账户、云端同步和多人共享

## 验收路径

1. 创建“锻炼 30 分钟”，开启周期任务，选择“每周 3 次”。
2. 点击三次“记录一次并结算”，确认进度从 0/3 到 3/3。
3. 确认第四次不能重复结算。
4. 创建一个“每天”任务，完成一次后确认当天不能再次结算。
5. 刷新页面，确认周期进度、金币和奖励记录仍保留。
