# Sprint 5 — 本周节奏

## 目标

将侧栏从静态导航升级为每日可见的行动提示：本周进度、下一件最值得推进的任务，以及愿望单的即时动力。

## 已包含

- 侧栏“本周节奏”自动读取当前周真实任务结算和 XP
- 任务完成、创建或兑换奖励后，节奏卡即时刷新
- 按任务类型与重要度推荐当前最值得推进的一项开放任务
- 显示当前首愿望与兑换所差金币
- 提供前往任务、奖励和周结算的快捷入口
- 侧栏品牌更新为 Questline / 人生工作台
- 统一使用自绘线性图标，替换文本符号

## 本轮不做

- AI 自动排程或强制推荐
- 自定义导航排序和主题切换
- 移动端独立侧栏交互
- 多账号与云端数据迁移

## 验收路径

1. 在任务页完成一项任务，确认侧栏已结算数量与 XP 立即更新。
2. 创建不同类型任务，确认 Boss 或高重要度任务优先显示为“下一件值得推进”。
3. 将奖励加入愿望单，确认节奏卡显示首愿望和剩余金币。
4. 确认侧栏导航、图标与 Questline 品牌在各页面一致。
