# Sprint 0：项目基础与浅色工作台

## 目标

建立一个可运行、可测试、可持续迭代的 Web 项目骨架，并呈现浅色工作台的静态基础界面。

## 范围

- 初始化 Next.js、TypeScript、Tailwind 与基础测试能力。
- 建立目录结构与环境变量示例。
- 创建浅色工作台布局：侧栏、顶部上下文区、主内容区。
- 提供 `/dashboard` 与 `/quests` 两个静态页面。
- 使用演示数据展示角色状态与今日任务，不实现结算逻辑。
- 将本目录的产品与架构文档纳入 README 导航。

## 不包含

- Supabase、认证、真实数据库
- 任务创建或完成的持久化
- XP 计算、金币结算、等级变更
- AI、社交、PWA、部署

## 验收标准

- 开发服务器能启动并展示两个页面。
- 页面在桌面与窄屏下可用。
- 浅色工作台符合 `docs/product/ui-direction.md`。
- TypeScript、lint、单元测试和生产构建均通过。
- 没有提交密钥、无关依赖或未说明的范围扩大。
