# Sprint 4A — 自定义奖励商城

## 目标

让用户把任务获得的金币兑换为自己预先定义的现实奖励，形成完成任务后的正向反馈闭环。

## 已包含

- 创建自定义奖励：名称、金币价格、是否可重复兑换
- 查看当前金币余额
- 兑换奖励并即时扣除金币
- 金币不足时禁止兑换，余额不会变为负数
- 一次性奖励兑换后不可再次兑换
- 保留最近兑换记录
- 浏览器刷新后保留奖励与兑换记录

## 本轮不做

- 编辑、删除、归档奖励
- 退款或撤销兑换
- 商城模板、图片、分类或推荐价格
- 共享奖励、付费、实物配送或外部支付

## 验收路径

1. 先完成若干任务，获得至少 1 枚金币。
2. 创建一个价格不高于余额的奖励并兑换。
3. 确认金币减少、最近兑换出现记录。
4. 创建高于余额的奖励，确认兑换按钮显示“金币不足”。
5. 创建一次性奖励并兑换，确认它不能再次兑换。
