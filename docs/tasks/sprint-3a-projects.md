# Sprint 3A — 副本基础与灵活关联

## 目标

让用户管理有明确终点的项目，并允许主线、副本、任务以任意顺序建立后再关联。

## 已包含

- 创建副本：名称、胜利条件、可选主线、可选目标日期
- 编辑主线名称与愿景
- 编辑副本名称、胜利条件、主线归属与目标日期
- 在创建或编辑非周期任务时选择、变更或清除所属副本
- 在创建或编辑任务时选择、变更或清除所属主线
- 已结算任务仍保留原有 XP、金币与奖励流水
- 副本页展示关联任务的完成数量与进度
- 今日页展示正在推进的副本摘要
- 浏览器本地数据兼容此前任务、主线与周期任务

## 本轮不做

- 删除、归档、手动关闭和副本奖励
- 里程碑、依赖、风险、负责人和协作
- 将周期任务关联到副本
- 编辑任务的计分属性或周期节奏
- 账户、云端同步和多人共享

## 验收路径

1. 先创建一项不关联主线的副本。
2. 再创建一条主线；回到副本页，点击“编辑”，关联该主线并保存。
3. 先创建一项不关联副本的普通任务；点击“编辑”，关联副本并保存。
4. 完成任务后再次编辑它的名称或归属，确认 XP、金币和结算记录不变。
5. 在主线页编辑主线名称或愿景，确认任务与副本显示同步为新名称。
6. 刷新页面，确认当前浏览器保留数据。
