# Sprint 3A — 副本基础

## 目标

让用户管理有明确终点的项目，并将一次性任务关联到该项目。

## 已包含

- 创建副本：名称、胜利条件、可选主线、可选目标日期
- 在创建非周期任务时选择所属副本
- 副本页展示关联任务的完成数量与进度
- 今日页展示正在推进的副本摘要
- 浏览器本地数据兼容此前任务、主线与周期任务

## 本轮不做

- 副本编辑、归档、手动关闭和副本奖励
- 里程碑、依赖、风险、负责人和协作
- 将周期任务关联到副本
- 账户、云端同步和多人共享

## 验收路径

1. 创建一条主线后，建立一个关联该主线的副本。
2. 为副本填写胜利条件与目标日期。
3. 创建一项非周期任务并关联到该副本。
4. 完成任务，确认副本页与今日页进度变化。
5. 刷新页面，确认当前浏览器保留数据。
