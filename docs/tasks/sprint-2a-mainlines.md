# Sprint 2A — 主线与任务关联

## 目标

让用户先定义长期方向（主线），再把具体任务关联到其中一条主线。

## 已包含

- 创建主线：名称与可选愿景
- 在创建任务时选择所属主线，也允许不关联
- 主线页显示关联任务的待推进与已完成数量
- 今日页展示正在推进的主线摘要
- 沿用浏览器本地测试数据，并兼容此前已保存的任务

## 本轮不做

- 主线编辑、归档或删除
- 主线权重、指标和目标日期
- 副本、重复任务、周结算
- 账户、云端同步和多人协作

## 验收路径

1. 在“主线”创建一条主线。
2. 在“任务”创建一条任务并选择该主线。
3. 确认任务列表与“今天”显示主线关联。
4. 完成任务后，确认主线页的完成数量更新。
5. 刷新页面，确认数据仍保留在当前浏览器。
